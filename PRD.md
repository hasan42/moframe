# MoFrame - Comic to Animation Converter

## Vision
Инструмент для превращения комиксов в анимированные видео с покадровым морфингом.

## Целевые пользователи
- Создатели контента для соцсетей
- Комикс-художники, желающие анимировать свои работы
- Энтузиасты визуального сторителлинга

## Функциональные требования

### Входные данные
- Поддержка форматов: JPG, PNG, WEBP, TIFF, PDF, CBR, CBZ
- Загрузка единичного файла или папки
- Авто-извлечение страниц из архивов CBR/CBZ
- PDF → изображения (разбиение по страницам)

### Обработка
- **Детекция панелей (frames):** автоматическое распознавание кадров комикса
- **Сегментация:** разделение страницы на отдельные панели
- **Упорядочивание:** определение последовательности чтения (L-to-R или R-to-L)
- **Морфинг:** плавный переход между панелями
  - Определение ключевых точек
- **Тайминг:** настройка длительности каждого перехода
- **Звук:** опциональная подложка аудио

### Выходные данные
- Формат: MP4 (H.264)
- Разрешение: 1920×1080 (Full HD)
- FPS: 24 или 30
- Соотношение сторон: сохранение пропорций с letterbox/pillarbox

### UI
- Веб-интерфейс (Streamlit / Gradio / Flask)
- Визуальный редактор последовательности
- Превью перед рендером
- Прогресс-бар обработки

## Технический стек

### Язык
- Python 3.10+

### Ключевые библиотеки
- **Обработка изображений:** Pillow, OpenCV, numpy
- **Морфинг:** face-morpher (или кастомная реализация на OpenCV)
- **Детекция панелей:** OpenCV (contours) или DL (YOLO / custom)
- **Видео:** moviepy, ffmpeg-python
- **UI:** Streamlit (быстрый старт) или Gradio
- **Архивы:** rarfile, zipfile
- **PDF:** pdf2image, PyMuPDF

## Архитектура

```
moframe/
├── core/
│   ├── loader.py          # Загрузка файлов
│   ├── panel_detector.py  # Детекция кадров
│   ├── morpher.py         # Морфинг между панелями
│   ├── sequencer.py       # Упорядочивание и тайминг
│   └── renderer.py        # Рендер в MP4
├── ui/
│   └── app.py             # Streamlit интерфейс
├── utils/
│   └── helpers.py
├── tests/
└── main.py
```

## Этапы разработки

### Phase 1: Core (Week 1-2)
- [ ] Загрузчик файлов (все форматы)

## Риски и решения
- **Сложный морфинг:** Начать с простого crossfade + масштабирование
- **Панели сложной формы:** Поддержка маски/ROI
- **Производительность:** Кэширование, опциональное уменьшение разрешения для preview
