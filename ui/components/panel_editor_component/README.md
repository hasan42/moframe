# MoFrame Panel Editor Component

Streamlit custom component for interactive panel editing.

## Build

```bash
npm install
npm run build
```

## Usage

```python
from panel_editor_component import render_panel_editor

updated_panels = render_panel_editor(image, panels, page_idx, key="editor_1")
```

## Development

```bash
npm run watch  # Auto-rebuild on changes
```
